# Project Overview

GiveawayZero is a web-based Giveaway Chess, or Antichess, AI platform. The project combines a polished chess analysis interface with a staged machine learning roadmap.

## What Is Giveaway Chess?

Giveaway Chess is a chess variant where the goal is usually to lose all pieces. Captures are mandatory when available, and the usual concept of checkmate is not the central objective. These rule differences create a game that feels familiar at first glance but requires a very different strategy.

## Product Concept

The first version of GiveawayZero will let users play Giveaway Chess against a simple AI bot in the browser. The interface should feel like a clean chess analysis board:

- Board-centered layout
- Clear move history
- Previous and next move navigation
- Legal move feedback
- Bot response after user moves
- Candidate move suggestions in later versions

The AI will begin simple so that the product is playable early. Later versions will use Lichess Antichess data and self-play methods to improve move quality.

## Why This Project?

GiveawayZero is intended as a portfolio-quality project that demonstrates:

- Full-stack application design
- Variant chess rule handling
- API design for turn-based games
- Data processing from public game databases
- Supervised machine learning for move prediction
- Model evaluation and iterative improvement
- A practical path toward self-play AI research

## Project Boundaries

The initial repository is intentionally documentation-first. The project will not start by building the full app or training models. Instead, the first milestone defines the structure, responsibilities, and roadmap so the implementation can grow in controlled phases.
