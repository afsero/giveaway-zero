# Training

This folder will contain data processing, model training, evaluation, and experiment configuration for GiveawayZero.

## Planned Role

The training workspace will support the AI roadmap from supervised learning on Lichess games through later self-play experiments.

## Planned Structure

```text
training/

  models/            PyTorch model definitions
  evaluation/        Metrics, match runners, and tournaments
  configs/           Experiment and training configuration
```

- `python-chess`
